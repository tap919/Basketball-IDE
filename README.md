# Basketball-IDE

A basketball-to-biotech IDE that enables biotech research through a translation layer of basketball concepts, based on 1-to-1 translation and synergy mapping of commonality and statistical accuracy.

## Features

- Load local LLM for AI-powered analysis
- VSCode extension support
- Basketball-to-Biotech translation layer
- Synergy mapping with statistical accuracy

## Data Files

- `regular_season_totals_2010_2024.csv` - NBA regular season statistics
- `play_off_totals_2010_2024.csv` - NBA playoff statistics

## Getting Started

```bash
# Install dependencies
bun install

# Start development server
bun run dev
```

Open [http://localhost:3000](http://localhost:3000) to see your application running.
